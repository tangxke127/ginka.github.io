如何初始化创建博客？

发表于2022-03-17|更新于2024-03-06
|字数总计:237|阅读时长:1分钟|阅读量:
  1. 安装node.js

  1. 安装淘宝镜像npm install -g cnpm –registry=https://registry.npm.tobao.org

  1. 安装hexo

cnpm install -g hexo-cli

查看版本：hexo -v

  1. 初始化一个博客

sudo hexo init (window中不用sudo:hexo init)

  1. 启动博客

hexo s

hexo n “我的文章一”

  1. 新建文章

hexo n “我的文章一”

查看文件：dir

  1. 部署博客到远端（如github）

新建仓库 名称是ginka.github.io

  1. 命令行中：安装插件

cnpm install —save hexo-deployer-git

安装hexo-deployer-git插件

win使用：npm install –save hexo-deployer-git

设置文件：写 仓库地址

部署到远端：hexo d（登录github账号）

  1. 刷新github仓库

10.清除缓存：hexo clean

》创建仓库

1.gitee中新建得到引入仓库指南

2.本地文件夹下 右键打开终端

//初始化

git init

//关联远程地址

git remote add origin https://gitee.com/gin963852/hexo_blog-cd1.git

git add .

git commit -m ‘初始化’

git push -u origin “master”

-—-建立仓库成功
